# B12-A10-Future-Box-Server

## Description
This is the backend server for the Future Box application.  
It handles user authentication, payment management, and other server-side functionalities. Built with **Node.js** and **Express**.

---

## Features
- User registration and login
- JWT authentication
- CRUD operations for payments
- PDF report generation
- Secure server-side operations

---

## Technologies Used
- Node.js
- Express.js
- MongoDB / Supabase / PostgreSQL (update according to your DB)
- JSON Web Token (JWT)
- Cors
- dotenv
